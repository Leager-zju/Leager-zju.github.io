C++ の 可变说明符(Mutable)

发表于 2023-03-02 更新于 2023-03-06 分类于 C++ Basic 阅读次数：

mutable 意为可变的，可以在非引用非常量非静态数据成员的声明中出现，允许被常量类对象修改。

类中的 mutable

先聊聊一个有趣的话题，关于 const 类成员函数有两大观点流派：

  1. bitwise constness：主张成员函数只有在不更改对象的任何非静态成员变量时才被认为是 const，即绝对常量； > 事实上这正是 C++ 对常量性的定义。
  2. logical constness：主张 const 成员函数可以修改所处理对象内部的某些 bits，仅当用户看不出的情况才行，即自适应常量；
