#正交向量

若两向量 a,b\mathbf{a}, \mathbf{b}a,b 正交，则必然有 aTb=bTa=0\mathbf{a}^T\mathbf{b} = \mathbf{b}^T\mathbf{a} = 0aTb=bTa=0

#正交子空间

若子空间 SSS 与子空间 TTT 正交，则 SSS 中任意向量与 TTT 中任意向量正交。

不难发现，R(A)R(A)R(A) 与 N(A)N(A)N(A) 是正交的，而这两个空间恰好都是 RnR^nRn 的子空间，更巧的是，其维数之和为 nnn。

我们称这两个子空间为 RnR^nRn 的正交补——它们将 RnR^nRn 划分为了两个正交的子空间。

同样的，C(A)C(A)C(A) 与 L(A)L(A)L(A) 也互为正交补。

两个互为正交补的子空间的维数之和，不能超过原空间的维数。

#投影问题

这一问题的几何解释为：如何在向量 a\mathbf{a}a 的方向上寻找与向量 b\mathbf{b}b 距离最近的一点。假设这一点对应的向量为 p\mathbf{p}p，则 e=b−p\mathbf{e}=\mathbf{b}-\mathbf{p}e=b−p 就是这一近似的误差。如图所示：

若令 p=x^a\mathbf{p} = \hat{x}\mathbf{a}p=x^a，因 e\mathbf{e}e 与 a\mathbf{a}a 正交，则有 aT(b−x^a)=0\mathbf{a}^T(\mathbf{b}-\hat{x}\mathbf{a}) = 0aT(b−x^a)=0，解得

x^=aTbaTap=aaTbaTa\hat{x} = \frac{\mathbf{a}^T\mathbf{b}}{\mathbf{a}^T\mathbf{a}} \\[2ex] \mathbf{p} = \mathbf{a}\frac{\mathbf{a}^T\mathbf{b}}{\mathbf{a}^T\mathbf{a}} x^=aTaaTb​p=aaTaaTb​

相当于把 b\mathbf{b}b 以某种操作投影到了 p\mathbf{p}p 上。若用投影矩阵 PPP 来描述这一操作，则有 p=Pb\mathbf{p} = P\mathbf{b}p=Pb，得

P=aaTaTaP = \frac{\mathbf{a}\mathbf{a}^T}{\mathbf{a}^T\mathbf{a}} P=aTaaaT​

这是一个对称矩阵（PT=PP^T=PPT=P），而且是幂等矩阵（P2=PP^2 = PP2=P）

#回到最开始的问题：如何求解无解方程组？

这个问题看似很离谱，实则我们可以用一个最优解 x^\mathbf{\hat{x}}x^ 去近似。对于方程 Ax=bA\mathbf{x} = \mathbf{b}Ax=b，如果其无解，说明 b\mathbf{b}b 不在 AAA 的列空间中。我们可以将 b\mathbf{b}b 投影到 C(A)C(A)C(A)，将问题转换为求解方程 Ax^=pA\mathbf{\hat{x}} = \mathbf{p}Ax^=p，这个解就是我们想求的近似最优解。

根据上面描述的，我们能够很轻易地得到该解，即

x^=(ATA)−1ATb\mathbf{\hat{x}} = (A^TA)^{-1}A^T\mathbf{b} x^=(ATA)−1ATb

#最小二乘法

应用投影矩阵求方程组最优解的方法，最常用于"最小二乘法"拟合曲线，如图：

已知数据点，求一直线方程，使该直线尽可能经过所有点——所有点到该直线的距离尽可能小。

有一个思路是：这些数据点大概率不会共线，也就是说，将这些数据点代入直线方程利用待定系数法求其参数时，组成的线性方程组大多数情况下是无解的，此时就要用到上面的"求无解方程组最优解"的思路。

#进入下一个议题：特征向量与特征值

将矩阵 AAA 与向量 x\mathbf{x}x 相乘当做是对向量的一种操作或者函数，输入 x\mathbf{x}x 而输出 AxA\mathbf{x}Ax 。特征向量即在特定的向量 x\mathbf{x}x 方向上输出的 AxA \mathbf{x}Ax 平行于 x\mathbf{x}x ，即为：

Ax=λxA\mathbf{x} = \lambda\mathbf{x} Ax=λx

其中 x\mathbf{x}x 为 AAA 的特征向量，λ\lambdaλ 为 AAA 的特征值。
